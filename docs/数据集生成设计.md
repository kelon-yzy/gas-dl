# V3.1 双通道 Waveform 数据集生成设计

> 创建日期：2026-05-14
> 状态：v0.1 设计稿（未实现）
> 适用项目：`V3_正式实验/`
> 上游实验设计：`docs/design.md`
> 输出数据集：`data/waveform_v3/`

## 1. 目的与范围

把 V3 单通道 waveform 升级到双通道，让仿真链路与专利硬件描述的两条独立物理链路严格对齐：

- **通道 1**：超声对射波形（TOF 链路）
- **通道 2**：光纤麦克风波形（声衰减链路 / 多次反射混响）

升级后的数据集用 `V3.1` 表示，与 V3.0 不兼容。

### 设计目标

| 目标 | 验收 |
| --- | --- |
| 硬件链路对应 | 通道 1 ↔ 超声对射；通道 2 ↔ 光纤麦克风；两通道物理独立 |
| 物理量解耦 | 通道 1 主要承载 TOF；通道 2 主要承载声衰减时间常数 tau |
| 信息可提取 | 从两通道波形可以分别估出 TOF / tau，与 ground truth 误差 < 5% |
| 方向性正确 | CO2 / H2O / H2 浓度变化时，两通道频谱 / 包络出现可观察的单调响应 |
| ML 与 DL 公平 | 两端都吃同一个双通道 waveform，差异只在抽取方式（信号处理 vs 端到端） |

### 不在本设计范围

- 多路径阵列（后续阶段）
- 换能器 impulse response、ring-down（后续阶段）
- 频散、群速度色散（后续阶段）
- 真实 DAQ / ADC 标定参数（calibration pending）

## 3. 物理模型

### 3.1 通道 1：超声对射波形

保留 V3.0 实现，只规范命名。

```text
emit(t) = A0 × Hanning(t, T_pulse) × sin(2π × f_c × t)
recv_us(t) = emit(t − TOF) × exp(−alpha(f_c) × L_m) + n_us(t)
```

承载信息：

- TOF 位置 → `c_sound` 反演
- 峰值幅度 → `alpha(f_c)` 在单频点的标量信息

### 3.2 通道 2：光纤麦克风波形

新建。光纤探头位于腔内固定点，记录发射 burst 后的腔内声场响应。

```text
p_field(t) = direct(t) + Σ reflection_k(t) + n_fm(t)
```

其中：

- `T_round = 2L / c_sound`
- `A_direct = exp(-alpha × L_direct)`
- `tau ≈ 1 / (alpha × c_sound)`
- 反射系数 `R = 0.5`（默认，calibration pending）

承载信息：

- 包络衰减时间常数 `tau`
- 反射峰间隔 `T_round`
- 反射峰数量与尾部能量统计

### 3.3 共用物理量

| 量 | 来源 |
| --- | --- |
| `c_sound` | `_hidden_sound_speed_v2(...)` |
| `alpha(f)` | `_hidden_attenuation_v2(..., f_hz=f_c)["alpha_true_v2"]` |
| 配气 + 工况 | 复用 V3.0 工况采样逻辑 |
| phase 序列 | `sim_common.phase_boundaries` |
| slow 通道 120 时步演化 | 复用现有动力学叠加 |

两通道在每个 timestep 共享同一组 `(gas, T, P, RH, L)`，天然时间对齐。

## 9. 方向性验证

由 `check_waveform_directionality.py` 输出 `quality/waveform_quality_summary.json`。

### 通道 1 必查项

- `L_m ↑ → peak_index ↑`
- `H2 ↑ → peak_index ↓`
- `CO2 ↑ → 峰幅度 ↓`
- `SNR > 40 dB`

### 通道 2 必查项

- `CO2 / H2O / H2 ↑ → tau ↓`
- `L_m ↑ → T_round ↑`
- `R ↑ → 包络尾巴更长`
- 前 3 次反射峰在大多数序列中可识别
- `tau` 估计与 `1/(alpha·c)` 强相关

## 12. 风险与边界

| 风险 | 应对 |
| --- | --- |
| `R`、`l_direct_factor` 等未校准 | 文档明确 `calibration_status: pending` |
| 光纤麦克风采样率未硬件确认 | 默认与超声同为 200 kHz |
| tau 物理范围超出测量窗 | 默认 10 ms 窗足够覆盖主要衰减段 |
| 数据体积大 | 必要时缩短通道 2 窗口 |
| 两通道公平性 | 两端都使用同一双通道原始波形 |

## 13. 与 V3.0 的差异

| 维度 | V3.0 单通道 | V3.1 双通道 |
| --- | --- | --- |
| 输入张量数 | 1 路 waveform + slow | 2 路 waveform + slow |
| 总磁盘 | 2.4 GB | ~7.2 GB |
| 主物理信息 | TOF 强 / alpha 弱 | TOF 强 + alpha 强 |
| ML 特征 | TOF / Amp 为主 | TOF + tau + 反射峰统计 |
| 兼容性 | 与 V3.1 不兼容 | 需重拟合 scaler、重生成 split |
| generation_seed | 旧 | `20260514` |
