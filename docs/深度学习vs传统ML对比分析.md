# 深度学习 vs 传统机器学习：模型表现对比分析

> 实验日期：2026-05-17 | 数据集：waveform_v3（10,000 序列，每序列 120 时间步）

## 2 深度学习落后传统 ML 的原因分析

### 2.1 数据集不对等

| | 传统ML (XGBoost→Ridge) | 深度学习 |
|---|---|---|
| **数据划分** | 2-way: 9,587 train / 2,411 test | 3-way: 7,028 train / 1,478 val / 1,494 test |
| **训练样本量** | 9,587 | 7,028（少 36%） |
| **数据过滤** | 5 阶段过滤（仅 stable 阶段 + 每混合物限 3 条） | 无过滤 |
| **验证集开销** | 无 | 1,478 样本（占总量 15%） |

传统 ML 训练集比 DL 多 2,559 个样本。对小数据集，这 36% 的差异不容忽视。此外传统 ML 的 5 阶段过滤（从 40,000 条记录中只保留约 12,000 条）剔除了不稳定阶段和冗余数据，而 DL 不加区分地使用全部 10,000 个序列。

### 2.2 特征表示方式差异（核心原因）

#### 输入信号对比

| 维度 | 传统 ML | 深度学习 |
|---|---|---|
| **时间步数** | 4 步（t=0, 70, 85, 100，手工选定） | 120 步（全序列） |
| **声学模态** | 4 个物理标量特征（TOF, Amp, f_peak, A_fft_max） | 原始 int16 波形（超声 1000 采样点 / 光纤麦克风 2000 采样点） |
| **光学模态** | 6 个派生物理特征（含 delta_I, A_NDIR 等差分/吸收量） | 慢变量 2 通道（V_NDIR_CH4, V_NDIR_CO2 原始电压时序） |
| **热学模态** | 1 个标量特征（V_TCS） | 慢变量 1 通道（V_TCS 原始电压时序） |
| **环境变量** | T_C, P_MPa, H_RH + 派生 13 变量（T_K, P_kPa, sound_speed 等） | 慢变量 3 通道（T_C, P_MPa, H_RH 原始时序） |
| **其他** | 无 | 慢变量 2 通道（L_m 光程, piston_position_m） |
| **数据结构** | **表格数据**，每行是独立样本，无时序关系 | **3D 张量 [B, T, C]**，保留时序结构 |

#### 核心差异

传统 ML 使用的 4 个声学特征（TOF, Amp, f_peak, A_fft_max）是通过**物理模型**从超声波形中提取的——峰值检测、FFT 频谱分析等。深度学习则直接输入 1000 个 int16 采样点，需要自己学习这些物理关系。

传统 ML 的光学特征还包含 `delta_I_CH4`（差分吸收电流）和 `A_NDIR_CH4`（归一化吸收）——这些是从 NDIR 电压信号中派生的物理量，比原始 V_NDIR 更接近真实的浓度映射。深度学习只有原始 NDIR 电压时序。

简而言之：**传统 ML 输入的是"答案的近似形式"，深度学习输入的是"问题本身"**。

### 2.3 时序利用的不对称

传统 ML 虽然选了 4 个关键时间步，但**没有时序建模能力**——每个时间步被当作独立的表格行处理，XGBoost/Ridge 不知道 t=0 和 t=70 之间的时间关系。

深度学习拥有完整的 120 步时序，CNN/LSTM/TCN 都可以捕捉时间依赖性。在信息量上深度学习占优，但代价是需要从更多噪声中提取有效信号，且 120 步中大量时间步可能对浓度预测没有信息增益（平稳阶段的数据几乎相同）。

传统 ML 的 4 个时间步是**经过物理理解挑选的**：baseline（t=0）、稳态初期（t=70）、稳态中期（t=85）、恢复期（t=100），每个时间步对应混合物扩散/稳定的不同阶段，具有较强的信息密度。

### 2.4 数据集规模不足以支撑端到端学习

10,000 个序列（DL 实际用 7,028 训练）对于需要从原始波形学习特征表示的深度学习来说偏小：

- XGBoost 等树模型对默认超参鲁棒，自带正则化（max_depth=5）
- 深度学习需要学习特征提取 + 时序建模 + 回归三层映射
- 声学波形编码器（AcousticWaveformEncoder）有 3 层 CNN + BatchNorm + 投影头，参数量不大但需要足够的样本来学习从 1000 采样点提取 TOF/Amp/f_peak 等价特征

### 2.5 深度学习超参未系统调优

| 模型 | batch_size | optimizer | weight_decay | LR scheduler |
|------|:---:|---|---|---|
| waveform_only | 8 | Adam | 0.0 | 无 |
| multimodal_fusion | 12 | Adam | 0.0 | 无 |
| TCN | 32 | Adam | 0.0001 | 无 |
| LSTM | 32 | Adam | 0.0001 | 无 |
| cnn1d_multimodal | 8 | AdamW | 0.01 | cosine_warmup |

各模型的 batch_size 从 8 到 32、weight_decay 从 0 到 0.01 不等，没有统一控制变量。cnn1d_multimodal 的 batch_size=8 + weight_decay=0.01 是较强的正则化组合，在小数据集上可能过度约束。

对比传统 ML：XGBoost(max_depth=5) + Ridge，固定参数网格，没有超参搜索但默认配置已经足够稳健。

### 2.6 传统 ML 的额外优势

1. **堆叠集成 + MC 扰动**：分支模型（PLS/XGBoost）在 5-fold stacking 框架下产生 OOF 预测，元模型（Ridge）融合；MC 扰动（对 T/P/H 加高斯噪声 × 24 样本）起到数据增强和预测不确定性估计的作用
2. **两阶段架构天然多模态**：声学/光学/热学各有独立分支模型，融合时元模型自动学习模态权重——这比 DL 的隐式特征拼接更显式地利用了模态结构
3. **无验证集开销**：省下的 1,478 个样本全部用于训练

## 3 改进方向

### 3.1 高优先级

1. **对齐数据过滤**
   - 对 DL 应用同样的 stable 阶段过滤 + 去重
   - 预期效果：减少噪声样本，可能提升 0.02-0.05 pp

2. **特征辅助融合**
   - 将传统 ML 的声学特征（TOF, Amp, f_peak, A_fft_max）作为额外通道拼接到慢变量中
   - 这些特征是免费的（已有提取代码），可以减轻波形编码器的学习负担
   - 或者：将 delta_I_CH4/CO2、A_NDIR_CH4/CO2 等光学派生特征也加入

3. **TPH 派生变量**
   - 传统 ML 加入 TPH 后 RMSE 从 1.396→0.633（-55%），但其收益不仅来自原始 T/P/H，还来自 13 个派生变量
   - 为 DL 补充 sound_speed、attenuation_alpha、P_dry_kPa 等物理意义明确的派生变量

### 3.2 中优先级

4. **系统性超参数搜索**
   - 统一 batch_size（建议 16 或 32）
   - learning_rate 网格搜索（1e-4 ~ 5e-3）
   - weight_decay 消融（0, 1e-4, 1e-3, 1e-2）
   - Gaussian noise / dropout 消融

5. **时间步采样策略**
   - 当前 time_window: "all" 使用全部 120 步，但很多时间步信息冗余
   - 实验使用传统 ML 的 4 个关键时间步作为对比
   - 或使用随机时间步采样作为数据增强

6. **增大数据集**
   - 仿真扩展生成更多混合物条件
   - 数据增强：时间维度微量扰动、MC 环境变量扰动（传统 ML 已有方案）

### 3.3 低优先级

7. **多种子实验**
   - 当前所有模型仅 seed=42 一次运行，无法评估方差
   - 建议 seed∈{42, 123, 456} 各跑一次，报告 mean±std

8. **公平对比实验设计**
   - 使用完全相同的数据划分（2-way 或统一 3-way）
   - 传统 ML 也使用 3-way 划分（或 DL 去掉验证集）
   - 确保测试集样本完全一致

## 4 关键结论

深度学习最佳模型（waveform_only, **0.640 pp**）与传统 ML 最佳模型（XGBoost→Ridge+TPH, **0.633 pp**）的差距仅 **1.1%**。

这一差距不应被解读为"深度学习不如传统 ML"，因为两者的输入信息量存在本质不对称：

| 不对称项 | 传统 ML 优势 |
|---|---|
| 声学特征 | 4 个物理模型提取的标量特征（TOF/Amp/f_peak/A_fft_max）vs 原始 1000 点波形 |
| 光学特征 | 6 个派生物理量（含差分吸收 delta_I）vs 2 个原始 NDIR 电压 |
| 环境特征 | 16 个变量（含 13 个派生）vs 3 个原始测量值 |
| 训练样本 | 9,587 vs 7,028（+36%） |

深度学习的端到端方案在**几乎零特征工程、输入信息远少于传统 ML** 的条件下达到了可比性能。这说明模型架构（AcousticWaveformEncoder + multimodal fusion）是有效的，当前差距主要来自特征信息的不对称和训练数据的规模差距，而非模型架构的根本缺陷。
