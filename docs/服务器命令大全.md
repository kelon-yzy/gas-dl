# 服务器命令大全

所有命令均在项目根目录 `V3_正式实验/` 下执行。

---

## 1. 深度学习训练

### 单次训练

```bash
# 带 CLI 进度条（本地交互使用）
python src/pipeline/train_deep.py --config configs/deep/<配置文件>.yaml

# 无 UI 模式（服务器后台推荐）
python src/pipeline/train_deep.py --config configs/deep/<配置文件>.yaml --no-ui
```

### 常用参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--config` | 配置文件路径 | `configs/deep/bounded_simplex_3task_loss_slow_branch.yaml` |
| `--epochs` | 覆盖配置中的 epoch 数 | `--epochs 200` |
| `--seed` | 覆盖随机种子 | `--seed 123` |
| `--output-root` | 覆盖输出根目录 | `--output-root /data/outputs` |
| `--no-ui` | 关闭 CLI 进度 UI | |
| `--resume` | 从检查点恢复训练 | `--resume outputs/.../last_checkpoint.pt` |
| `--checkpoint-every` | 每 N epoch 保存检查点 | `--checkpoint-every 10` |
| `--no-resume-rng` | 恢复训练时不恢复 RNG 状态 | |
| `--stop-after-epoch` | 训练到指定 epoch 后停止 | `--stop-after-epoch 50` |

### 断点续训

```bash
# 从上次中断处恢复
python src/pipeline/train_deep.py \
  --config configs/deep/<配置文件>.yaml \
  --resume outputs/exp02_deep_e2e/<run_name>/last_checkpoint.pt \
  --no-ui
```

### 多实验并行

多个实验可同时运行（输出目录不同即可），取决于 GPU 显存。

```bash
# 终端 1
python src/pipeline/train_deep.py --config configs/deep/bounded_simplex_3task_loss_slow_branch.yaml --no-ui &

# 终端 2
python src/pipeline/train_deep.py --config configs/deep/bounded_simplex_3task_uw_slow_branch.yaml --no-ui &
```

指定不同 GPU（多卡时）：在 yaml 中修改 `training.device: cuda:0` / `cuda:1`。

### 后台运行（nohup）

```bash
nohup python src/pipeline/train_deep.py \
  --config configs/deep/<配置文件>.yaml --no-ui \
  > logs/<run_name>.log 2>&1 &
```

## 3. 可视化与分析

### 训练曲线

```bash
# 绘制某个目录下所有 run 的训练曲线
python src/pipeline/plot_deep_training_curves.py --root outputs/exp02_deep_e2e

# 绘制单个 run
python src/pipeline/plot_deep_training_curves.py \
  --root outputs/exp02_deep_e2e/<run_name> \
  --output-dir outputs/deep_training_curves/<run_name>
```

| 参数 | 说明 |
|------|------|
| `--root` | 扫描的根目录（递归查找 train_log.csv） |
| `--output-dir` | 图片输出目录 |
| `--formats` | 输出格式：`png` `svg` `pdf`（可多选） |
| `--dpi` | 图片 DPI |

### 单次运行详细分析

```bash
python src/pipeline/plot_deep_run_analysis.py \
  --run-dir outputs/exp02_deep_e2e/<run_name> \
  --output-dir outputs/analysis/<run_name>
```

| 参数 | 说明 |
|------|------|
| `--run-dir` | **必需**，单个 run 的输出目录 |
| `--output-dir` | 分析图片输出目录 |
| `--formats` | 输出格式 |
| `--dpi` | 图片 DPI |

### 灵敏度分析

```bash
python src/pipeline/sensitivity_scan.py \
  --predictions outputs/exp02_deep_e2e/<run_name>/predictions.csv \
  --condition-grid ../../data/waveform_v3/condition_grid.csv \
  --output-dir outputs/sensitivity/<run_name>
```

| 参数 | 说明 |
|------|------|
| `--predictions` | **必需**，预测结果 CSV |
| `--condition-grid` | **必需**，实验条件网格 |
| `--output-dir` | **必需**，输出目录 |
| `--model-name` | 图表中的模型名称 |

## 8. 常用组合命令

```bash
# 训练完成后一键出图
python src/pipeline/train_deep.py --config configs/deep/<配置>.yaml --no-ui && \
python src/pipeline/plot_deep_run_analysis.py --run-dir outputs/exp02_deep_e2e/<run_name>

# 后台训练 + 日志记录 + 完成后出图
nohup bash -c '
  python src/pipeline/train_deep.py --config configs/deep/<配置>.yaml --no-ui && \
  python src/pipeline/plot_deep_run_analysis.py --run-dir outputs/exp02_deep_e2e/<run_name>
' > logs/<run_name>.log 2>&1 &

# 查看训练日志（实时跟踪）
tail -f logs/<run_name>.log

# 查看 GPU 使用（每 2 秒刷新）
watch -n 2 nvidia-smi
```
